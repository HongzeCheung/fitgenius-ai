## 📱 响应式断点说明

```css
/* Tailwind 断点 */
sm: 640px   /* 小屏手机 */
md: 768px   /* 平板 */
lg: 1024px  /* 笔记本 - 我们的主要断点 ✅ */
xl: 1280px  /* 桌面 */
2xl: 1536px /* 大屏 */
```

**使用示例**:
```typescript
// 移动端: 1 列, PC 端: 2 列
className="grid grid-cols-1 lg:grid-cols-2 gap-6"

// 移动端: 隐藏, PC 端: 显示
className="hidden lg:block"

// 移动端: max-w-md (448px), PC 端: max-w-7xl (1280px)
className="max-w-md lg:max-w-7xl"
```

## 🚀 代码实现亮点

### 1. 智能侧边栏显示
```typescript
{activeView === 'dashboard' && (
  <aside className="hidden lg:block lg:col-span-4 space-y-6">
    {/* 侧边栏内容 */}
  </aside>
)}
```
- 仅在 dashboard 视图显示
- 移动端自动隐藏
- PC 端占 4 列

### 2. 动态数据计算
```typescript
const thisWeek = sortedLogs.filter(log => {
  const logDate = new Date(log.date);
  const weekAgo = new Date();
  weekAgo.setDate(weekAgo.getDate() - 7);
  return logDate >= weekAgo;
});
```
- 实时计算本周数据
- 无需额外状态管理
- 性能优化 (使用 IIFE)

### 3. 响应式网格
```typescript
<div className="lg:grid lg:grid-cols-12 lg:gap-6">
  <div className="lg:col-span-8">{/* 主内容 */}</div>
  <aside className="lg:col-span-4">{/* 侧边栏 */}</aside>
</div>
```
- 12 列网格系统
- 8:4 黄金比例
- 移动端自动堆叠
